# Wizard Scorekeeper

A fast, mobile-friendly scorekeeper for the card game Wizard, built to make round tracking clean, clear, and fun at the table.

## Stack

- Vite
- React
- TypeScript
- Material UI (MUI)
- Jotai with atomWithStorage for persistent game state

## Credits

Created by Gregory Rutherford.

Pixel art assets were created using Aseprite.

## Development

```bash
npm install
npm run dev
```

## Build

```bash
npm run build
```

## Deploy to GitHub Pages

This repository is configured to deploy automatically with GitHub Actions.

1. Push this project to a GitHub repository named `wizard-scorekeeper`.
2. In GitHub, go to Settings > Pages.
3. Under Build and deployment, set Source to `GitHub Actions`.
4. Push to `main` to trigger deployment.

Your app will be available at:

`https://<your-github-username>.github.io/wizard-scorekeeper/`
